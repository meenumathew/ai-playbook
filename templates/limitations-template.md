# Known Limitations

System-wide limitations that affect what can and cannot be built. Check here before assuming a capability exists.

New developer? Read this before building: undocumented limitations become someone else's surprise.

---

## When to Record

Add an entry when:

- A feature explicitly does not support something users might expect
- A performance ceiling is known but not enforced
- An integration only works with specific providers/versions
- A workaround is required for a known gap
- A limitation was discovered during implementation

Remove an entry when the limitation is resolved: add a commit reference.
