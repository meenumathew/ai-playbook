# Documentation Index

Use this page to enter the docs by reader need. The repository landing page remains [../README.md](../README.md); this file is the docs-folder index. The documentation follows strict Diataxis: every page has one primary job. When a page needs another kind of content, link to that page instead of mixing modes.

## Authoring Rule

Before adding or rewriting a page, choose exactly one primary quadrant. If the page starts serving a second reader need, split it or link to the right existing page. See [Documentation Guide](docs-guide.md) for the full authoring contract.
